# VoiceCraft — Text to Speech Converter

A modern and responsive Text-to-Speech web application built using HTML, CSS, and JavaScript with the Web Speech API.

## ✨ Features

- 🔊 Convert text into speech instantly
- 🎤 Multiple voice selection
- ⚡ Adjustable speech rate
- 🎚 Adjustable pitch control
- 🔉 Volume control
- ⏯ Play, Pause, Resume, and Stop controls
- 📋 Copy text to clipboard
- 📥 Download text as `.txt`
- 🌙 Dark / Light mode
- 📱 Fully responsive modern UI
- 🎵 Animated voice visualizer
- ⌨ Keyboard shortcuts support

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/darshanworks/text-to-speech-converter.git
```

Open the project folder:

```bash
cd voicecraft-text-to-speech
```

Run the project by opening:

```bash
index.html
```
